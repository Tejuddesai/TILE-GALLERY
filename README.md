We are designing a web application for Inerier Duniya. This application will enable any users to purchase the Tiles.  Thus, we are developing a website which goes hand in hand with new age                                                                                                                                                                                                         technology, user friendliness, informative and time saving. 

Features
User-Friendly Interface: Easy navigation and search capabilities to browse through various tile categories.
Dynamic Filters: Filter tiles by size, material, price, and color.
Product Details: Detailed descriptions, images, and customer reviews for each tile.
Shopping Cart: Add, remove, or update items in the cart with real-time price calculation.
Secure Checkout: Multiple payment options and secure payment gateways.
Admin Panel: Manage products, orders, and users efficiently.

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: C#,ASP.NET
Database: MySQL
